# Decision 2 ‚Äî `idgen` public API & prefix placement

**Decision.** `idgen` owns the format grammar in both directions and stays a
pure function of its inputs (no clock). The prefix is a parameter on mint and is
stripped-and-ignored on decode.

```go
package idgen

// Epoch is the zero point: 2026-01-01 00:00:00 UTC, constructed with
// time.Date(..., time.UTC) so it carries no monotonic reading.
var Epoch = time.Date(2026, time.January, 1, 0, 0, 0, 0, time.UTC)

// ErrInvalidID wraps the error TimeOf returns for a malformed id.
var ErrInvalidID = errors.New("invalid id")

// MintAt returns "<prefix>-XXXX-XXXX" for the given instant. Instants
// before Epoch are clamped to Epoch. The caller guarantees prefix is a
// non-empty run of letters/digits (cli validates); MintAt does not
// re-validate.
func MintAt(prefix string, t time.Time) string

// TimeOf inverts the body of any "<prefix>-XXXX-XXXX" id to the instant
// it was minted from. The prefix is accepted but ignored ‚Äî the instant
// lives entirely in the body ‚Äî so an id of any prefix decodes. Returns
// an error wrapping ErrInvalidID when id is not canonical.
func TimeOf(id string) (time.Time, error)
```

Body math, identical to the reference except the epoch:
`n = (ms¬∑0x9E3779B1 + 0xC0FFEE) mod 36‚Å∏` via `math/big`, uppercase base-36,
zero-padded to 8, split 4-4. `TimeOf` inverts with the precomputed modular
inverse of the multiplier; package `init` panics if multiplier and `36‚Å∏` ever
lose coprimality (fail-loud ‚Äî every existing id would otherwise be
irrecoverable). Decode grammar: `^[A-Za-z0-9]+-([0-9A-Z]{4})-([0-9A-Z]{4})$`.

- Prefix is a **parameter**, not baked in: `idgen` knows the grammar (body,
  base-36, dash split); the prefix string is supplied by the caller.
- Prefix **validation lives in `cli`** (validate at the flag boundary). A prefix
  containing `-` would corrupt the decode grammar; cli's letters/digits-only
  rule prevents it. `MintAt` documents and trusts the precondition.
- `idgen` holds **no clock** ‚Äî the instant is always passed in; the `-n` wait
  concern moves to `cli`.

**Rejected.**
- *`idgen` deals only in the 8-char body; `cli` glues on `prefix-` and the
  dashes.* Leaks base-36/4-4 format into `cli` and splits decode-parsing across
  packages.
- *A `New`/`Generator` clock-carrying API.* Clock moves out to `cli`, so
  `idgen` needs neither. Kept `TimeOf` (reads as "the time of this id") over
  `Decode`.

**Verification.**
- `R-WH5F-QJYS` ‚Äî Round-trip **randomized property sweep**: an ordinary `Test`
  (not a Go `Fuzz` target, so it runs in full on every `go test -race ./...`) draws
  a large PRNG-seeded sample of `ms ‚àà [0, 36‚Å∏)` and asserts
  `TimeOf(MintAt(p, Epoch+ms)) == Epoch+ms` (truncated to ms) across several valid
  prefixes. The discriminating property is breadth: a fixed handful of cases (or a
  bare `Fuzz` seed corpus, which the plain gate never expands) does not satisfy
  this id ‚Äî the sample size must be large (hundreds+) each run.
- `R-WIDC-4BPH` ‚Äî Golden vector at `Epoch` (ms 0): `Epoch` ‚Üí its exact
  `R-XXXX-XXXX` string, independently derived ‚Äî pins the affine **offset** constant
  at offset zero. This vector alone does **not** lock the epoch value: the ms-0
  image is invariant under moving `Epoch` (feeding the `Epoch` symbol always yields
  offset 0), so a test that inputs `Epoch` cannot fail on an epoch regression. The
  epoch lock is R-WJL8.
- `R-WJL8-I3G6` ‚Äî Golden vector at a **fixed absolute UTC instant** a known
  non-zero duration after 2026-01-01 (written as a literal civil time, independent
  of the `Epoch` symbol) ‚Üí its exact string, independently derived. Because the id
  encodes `instant ‚àí Epoch`, this locks **both** the affine multiplier / 4-4 split
  **and** the 2026 epoch: redefining `Epoch` shifts the offset (or clamps it) and
  fails this vector. Inputting `Epoch + offset` instead of a literal absolute
  instant would not lock the epoch and does not satisfy this id.
- `R-WKT4-VV6V` ‚Äî Padding: small ms values still yield 8 body chars.
- `R-WM11-9MXK` ‚Äî Clamping: instants before Epoch encode as Epoch (ms 0).
- `R-WN8X-NEO9` ‚Äî Prefix-agnostic decode: ids with prefixes `R`, `S`, `SPEC` all
  decode to the same instant for the same body.
- `R-WPOQ-EY5N` ‚Äî Malformed input: bad bodies/shapes return an error wrapping
  `ErrInvalidID`.
- `R-WQWM-SPWC` ‚Äî `TimeOf` robustness sweep: an ordinary `Test` (not a `Fuzz`
  target) feeds a large PRNG-seeded sample of arbitrary/adversarial strings (random
  bytes, near-valid shapes, wrong lengths, bad runes) and asserts none panic ‚Äî each
  returns either an `ErrInvalidID`-wrapping error or a valid time. Breadth is the
  discriminating property: a fixed seed list does not satisfy this id.
- `R-WS4J-6HN1` ‚Äî Coprimality fail-loud: package `init` panics if the multiplier
  and `36‚Å∏` ever lose coprimality (every existing id would otherwise be
  irrecoverable).
